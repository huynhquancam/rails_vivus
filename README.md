Rails Vivus
=========================

Rails Vivus assets https://github.com/maxwellito/vivus

##Installation

```sh
gem install rails_vivus
```

or add this to your Gemfile

```ruby
gem 'rails_vivus'
```

then bundle it.

##Usage

Add this to your `application.js`

```javascript
//= require vivus
//or
//= require vivus.min
```

Happy SVG!!!
